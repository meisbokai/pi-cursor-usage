All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-07-02

### Added

- Pi footer usage monitor for Cursor provider sessions.
- Auto + Composer and API usage percentages matching the Cursor portal.
- Dashboard API auth via local Cursor Desktop session (`GetCurrentPeriodUsage`).
- Admin API auth via `CURSOR_ADMIN_API_KEY` (`POST /teams/spend`).
- `/cursor-usage` command for detailed usage output.
